## Image Naming Convention
- Use lowercase with hyphens
- Keep names descriptive
- Use .jpg or .png format
- Recommended size: 1200x800px or similar

## After Adding Images
Update the HTML slides to reference them like:
```html
<img src="images/owl-device-front.jpg" alt="Meeting Owl 3 device" style="max-width: 600px;">
```
